# Dynamic Resource Editor (2.1) for MTA:SA

⚠️ **Disclaimer:** This tool was 100% generated by AI (Google's Gemini).

Dynamic Resource Editor (DRE) is a professional-grade, real-time In-Game IDE and file manager for Multi Theft Auto: San Andreas. It brings a VS Code-like development experience directly into the game, allowing developers and mappers to create, edit, manage, and inspect game elements in real-time.

## 🚀 Key Features

* **🎯 Select Object Mode:** Capture object data directly in-game. Toggle the mode, point your crosshair at any object, and click to inspect its properties (Position, Rotation, Model ID, etc.) in a new inspector tab.
* **📂 Enhanced File Management:** 
    * **Multi-Selection:** Hold `CTRL` to select multiple files in the explorer.
    * **Batch Copying:** Copy multiple files simultaneously to any resource with a simple interface.
* **VS Code Look & Feel:** Modern dark-mode interface with robust syntax highlighting.
* **Tab-System:** Edit multiple files simultaneously with an intuitive tabbed interface.
* **Smart File Manager:** Create, delete, rename, or copy files. The `meta.xml` is automatically updated and maintained.
* **Backup & Recovery:** Automatic timestamped backups and a built-in "Restore" feature.
* **Admin Logs:** Detailed history of all file operations stored in `logs.db`.
* **Template Injector:** Easily insert code snippets (e.g., Jump Markers, Teleports, SlowMo) directly into your scripts.
* **Auto Updater:** Keep the tool updated effortlessly using the `/updateeditor` command.

## ⚠️ Security & Requirements

**CRITICAL:** This tool provides read/write access to your server's file system.

* **ACL Permissions:** Ensure the resource is added to the "Admin" group in your `acl.xml`:
  `<object name="resource.DynamicResourceEditor" />`
* **Security:** Only logged-in Admins have access to the UI. Unauthorized users are blocked automatically.

## 📥 Installation

1. Clone the repository or download as `.zip`.
2. Extract the folder into your server's `resources` directory.
3. Ensure the folder name is exactly `DynamicResourceEditor`.
4. Add the resource to your `acl.xml` (see Security Warning).
5. Start the server, type `refresh`, and then `start DynamicResourceEditor`.

## 🎮 Usage

* **Toggle Editor:** Press `F2` in-game to open/close the IDE.
* **File Management:** * **Left-click:** Select a file.
    * **Right-click:** Open the Context Menu (Copy, Rename, Delete).
* **Editing:** Double-click a file to open it in a new tab.
* **Saving:** Click "Save File" in the toolbar.
* **Backups:** Click the **📜 (Activity Log)** icon, locate the file operation, and click "Restore" to revert to a previous version.
* **Templates:** Use the "Templates" dropdown to inject code snippets at your cursor position.

*Note: This editor is in active development. If you encounter errors, check your server console for logs!*
